# Encryption Demo Toolkit – National Award Winner

![Python](https://img.shields.io/badge/python-3.8%2B-blue)
![Award](https://img.shields.io/badge/Award-2nd%20Place%20National-orange)

Interactive Python tool demonstrating classical encryption algorithms with a Tkinter GUI.

**Awarded 2nd Place Nationally** at the Chi Beta Phi Scientific Honor Society Conference (October 2025) for clear explanation and live demo of encryption concepts.

### Features
- Caesar Cipher (shift encryption)
- Vigenère Cipher (polyalphabetic)
- Full RSA implementation (key generation, encrypt/decrypt)
- Real-time GUI for testing messages
